INSTRUCCIONES PARA COLOCAR ÍCONOS:

1. Ícono de la Aplicación (icon.png):
   - Ruta: static/description/icon.png
   - Tamaño recomendado: 256x256 px o 128x128 px
   - Formato: PNG con transparencia
   - Se muestra en la sección de Aplicaciones

2. Banner de la Aplicación (banner.png - Opcional):
   - Ruta: static/description/banner.png
   - Tamaño recomendado: 1200x630 px
   - Formato: PNG
   - Se muestra en la página de detalles del módulo

3. Pasos para agregar tu ícono:
   a) Copia tu ícono a esta carpeta (static/description/)
   b) Renómbralo como "icon.png"
   c) El manifest ya está configurado para buscarlo aquí

El manifest apunta a:
- 'web_icon': 'spt_cashflow_tool,static/description/icon.png'

¡Listo para cargar tu ícono!
